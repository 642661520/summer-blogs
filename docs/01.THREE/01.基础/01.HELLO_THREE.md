# 坐标辅助器

引入坐标辅助器后可以看到坐标轴的位置，方便调试

需要注意的是 红色代表 X 轴，绿色代表 Y 轴，蓝色代表 Z 轴

:::code-view
AxesHelper
:::

<br/>

# 控制器

引入控制器后就可以通过鼠标控制视角，滚轮缩放

需要注意的是开启阻尼效果后，必须在渲染循环中调用更新函数

```js
import { OrbitControls } from 'three/examples/jsm/Addons.js';
```
